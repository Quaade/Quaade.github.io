Prototype Fighter
An Experiment with pseudo-3d graphics using sprite stacking

2018
Prototype
Solo


This project was an experiment in using voxel software to create 2D isometric sprite animations that could be viewed from any angle. By layering each layer of the voxel based model on top of each other with a few pixels of offset, a fairly convincing 2d pixel art effect could be achieved, without needing to redraw every frame of every animation from multiple angles. This required writing a fairly comprehensive library of helper functions to simplify the import and rendering process.
